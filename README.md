# Description
Cripto currency made in agda

# Build with nix
Install nix and run this command:
```bash
$ nix-build
```

# Latex File
Some latex files (agda.sty and .tex) are compiled in gh-pages.
Execute this command to get them
```bash
$ bash copy-latex-ghpages.bash
```
